# Frontend — Admin Dashboard

## Screens
- [ ] Dashboard — KPI cards, AreaChart (revenue), PieChart (cylinder distribution), Recent Orders table, Activity feed
- [ ] Customers — Stat cards, search/filter toolbar, table with side profile panel, View/Suspend actions
- [ ] Vendors — Tab bar (All/Pending/Suspended), search, table with detail panel, Approve/Reject
- [ ] Riders — Stat cards, tab bar, table with detail panel, Assign/Suspend
- [ ] Orders — Filter bar, status tabs, table with timeline detail panel, Invoice/Refund
- [ ] Payments — Stat cards, tabs (Transactions/Refunds/Settlements/Failed), table with payment detail panel
- [ ] Analytics — Date range selector, KPI cards, LineChart, BarChart, Top Vendors
- [ ] Admin Users — Manage admin accounts, roles, status
- [ ] Refunds — Refund requests list, approve/reject actions, history
- [ ] Support Tickets — Ticket list, search, status, resolve action
- [ ] Audit Logs — Timeline of admin actions with filters by admin, resource, action, date range
- [ ] System Health — API status, database, queue, active users
- [ ] Notifications — Announcements list, system alerts, notification channel toggles
- [ ] Settings — Platform settings, feature toggles, admin users, security settings

## Components
- [ ] KPIStatCard — Metric display with label, value, trend, period
- [ ] AnalyticsChart — Reusable chart component (Line, Bar, Pie, Area)
- [ ] DataTable — Sortable, filterable, paginated table
- [ ] FilterToolbar — Search input, status/date/role dropdowns, clear
- [ ] SearchInput — Debounced search field
- [ ] StatusBadge — Color-coded status pill
- [ ] DetailDrawer — Slide-out panel for row details
- [ ] AuditLogTable — Timestamp, admin, action, resource, details columns
- [ ] NotificationPanel — System alerts, announcements, mark-as-read
- [ ] SettingsCard — Grouped setting field with edit action
