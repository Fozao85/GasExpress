# Sprint 02 — Customer Discovery

## Objective
Enable customers to discover nearby LPG vendors, compare available gas products, view vendor information, and select a vendor for ordering.

## Backend
- Vendors API (list, search, nearby)
- Categories API
- Product availability
- Location-based matching
- Vendor ranking (distance, speed, price, rating)
- Operating hours validation

## Frontend
- Home screen
- Search vendors
- Vendor listing (with sort/filter)
- Vendor details
- Promotions / offers

## Deliverables
Customers can browse and compare vendors with location-aware results.

## Definition of Done
- Vendor listing loads with real location-aware data
- Search, sort, and filter all work correctly
- Vendor detail page shows all info including availability
- Closed/out-of-stock vendors are handled appropriately
- UI matches high-fi designs
