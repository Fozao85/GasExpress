# Sprint 03 Acceptance Criteria

## Cart

Customer can:

- Add products
- Change quantity
- Remove products
- View total price

---

## Checkout

Customer can:

- Select address
- Select payment method
- Confirm order

---

## Order Creation

System creates:

- Order record
- Order items
- Payment record

---

## Validation

System prevents:

- Empty cart checkout
- Invalid addresses
- Invalid payment requests

---

## UI

Screens must:

- Match High-Fi designs
- Have loading states
- Have error states
- Have success feedback

---

# Definition Of Success

Sprint 03 is complete when a customer can successfully place an LPG gas order from a selected vendor.
