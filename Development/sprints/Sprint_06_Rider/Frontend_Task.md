# Frontend — Rider

## Screens
- [ ] Identity Verification — Submit ID, driver's license, vehicle documents
- [ ] Pending Approval — Waiting screen with estimated review time
- [ ] Dashboard — Stats (online time, deliveries today, earnings today), online/offline toggle, quick actions
- [ ] Jobs — Available jobs list, active job card, job detail with map + customer info
- [ ] Navigation — Turn-by-turn to pickup/delivery
- [ ] Earnings — Daily/weekly/monthly breakdown, chart, payout history
- [ ] History — Completed deliveries list with filters
- [ ] Profile — Personal info, vehicle details, verification status, settings
- [ ] Notifications — Job offers, status updates, system alerts
- [ ] Settings — Notification prefs, privacy, help, logout

## Components
- [ ] JobCard — Pickup/delivery addresses, distance, payout, accept/reject actions
- [ ] ActiveDeliveryCard — Current job summary with status and progress
- [ ] EarningsCard — Amount earned with period label and trend
- [ ] DeliveryTimeline — Multi-step status stepper (Accept → Arrived Vendor → Picked Up → Arrived Customer → Delivered)
- [ ] NavigationMap — Map with route line, vendor pin, customer pin, rider pin
- [ ] RiderStatusToggle — Online/Offline switch with visual indicator
- [ ] EarningsChart — Bar/line chart with period selector
- [ ] HistoryCard — Delivery summary, date, amount, reorder
- [ ] ProfileCard — Photo, name, vehicle, rating, verification badge
