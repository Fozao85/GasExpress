# Sprint 01 — Authentication & User Management

## Objective
Build a secure authentication system for Customers, Vendors, Riders, and Administrators.

## Backend
- User Registration
- Login
- OTP Verification
- Password Reset
- JWT Authentication
- Refresh Tokens
- Role-Based Access Control

## Frontend
- Customer: Splash, Welcome, Register, Login, OTP, Forgot/Reset Password
- Vendor: Login, Register, OTP
- Rider: Login, Register, OTP
- Admin: Login

## APIs
- Authentication APIs
- User APIs

## Deliverables
Complete authentication flow across all four apps.

## Definition of Done
- All authentication APIs work
- JWT authentication implemented
- Role protection working
- All authentication screens completed
